# Jackson Lab Antibody Tracker

Standalone HTML antibody inventory tracker for the Jackson Lab.

## Usage
Open `antibody-tracker.html` in any browser. Data is saved in localStorage.

## Publishing to GitHub Pages
1. Make edits in `antibody-tracker.html`
2. Click **⬆ Export for Web** in the tracker controls bar
3. Replace `index.html` in this repo with the downloaded file
4. `git add index.html && git commit -m "update tracker data" && git push`
5. Changes appear at your GitHub Pages URL within ~30 seconds
